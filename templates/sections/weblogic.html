Oracle Weblogic Applications

Oracle WebLogic is a popular platform upon which to deploy J2EE applications. WebLogic has the ability to broker JDBC connections to database services on behalf of applications, but it order to facilitate this it requires credentials with which to authenticate to the server. These credentials are usually stored on the WebSphere server which, if compromised, could provide an attacker with a means to retrieve them. These credentials can then be exploited to connect to the database and extract any data that the account has access to. As with any other application credential, enforcing least-privilege policies and rotating credentials, if possible, is the standard defense against such a threat, but it is also possible to remove hard-coded credentials entirely and store them in the CyberArk Secure Vault, with programmatic retrieval via the Application Identity Manager API.
